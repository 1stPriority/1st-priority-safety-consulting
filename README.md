# 1st Priority Safety Consulting — Website

This repository contains a simple static website intended for GitHub Pages or any static hosting.

## Structure
- index.html — homepage
- about.html, services.html, sales.html, booking.html, contact.html
- css/style.css
- CNAME — add your custom domain (optional)
- INSTRUCTIONS.md — step-by-step deploy instructions

## How to use
1. Edit the HTML/CSS files to match your content.
2. Deploy to GitHub Pages (see INSTRUCTIONS.md) or upload to any static host.
